Nightly Tests¶

The following regression tests are run nightly with MFiX-Exa. The plotfiles generated in each night’s test are compared with the benchmark plotfiles using the AMReX fcompare utility to compare the mesh data and particle_compare to compare the particle data.

The results of these tests can be found at https://ccse.lbl.gov/pub/RegressionTesting/MFIX-Exa/

Below Ng = number of grids, Npa = number of particles, Np = number of MPI ranks.

“Auto” means the particles were generated automatically with the random number generator; if “Auto” is not specified the particle data were read in from “particle_input.dat”

These first tests have both fluid and particles and are run in rectangular geometries; they all use drag type “BVK2”.
