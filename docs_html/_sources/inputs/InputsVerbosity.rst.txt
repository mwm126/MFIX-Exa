Verbosity
=========

The following inputs must be preceded by "mfix."

+----------------------+-----------------------------------------------------------------------+-------------+--------------+
|                      | Description                                                           |   Type      | Default      |
+======================+=======================================================================+=============+==============+
| verbose              |  Verbosity in MFiX-Exa routines                                       |    Int      |   0          |
+----------------------+-----------------------------------------------------------------------+-------------+--------------+
| ooo_debug            |  If true then print the name of the routine we are in                 |   Bool      | False        |
+----------------------+-----------------------------------------------------------------------+-------------+--------------+
