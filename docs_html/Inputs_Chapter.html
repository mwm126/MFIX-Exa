Run-time InputsÂ¶

Run-time inputs are used to control the simulation setup such as the grid, solver settings, geometry, boundary conditions etc. These <key> = <value(s)> pairs are saved in a text file, with any file name, that is passed to the MFiX-Exa executable at run time:

./mfix my_project.inputs

Note

Traditionally, input files have been named inputs. However, it is recommended that you provide a descriptive name with an extension of .inputs such as fluid_bed.inputs
